SumType

A tool for typing fully formatted transliterations of Sumerian, by Victor Voermans, 2024.

Input
Output

You can copy the output and paste it into a program like Word. Make sure your settings are correct so the formatting doesn’t get lost.

Numbers are turned into special subscript characters to ensure that determinatives like lu₂ are rendered correctly.
